# ⚡ Electricity Saving Guide

A smart web-based application that helps users optimize their electricity consumption and reduce monthly bills using algorithmic strategies.

---

## 🚀 Overview

This project analyzes a user's electricity usage and suggests optimized plans to reduce consumption and shift to a lower tariff slab.

It uses multiple algorithmic approaches to generate different strategies for saving electricity and money.

---

## ✨ Features

- 🔢 Input monthly electricity units
- ⚡ Select target electricity slab
- 🏠 Choose household appliances with quantity
- 🧠 Multiple optimization algorithms:
  - Greedy Algorithm
  - Balanced Distribution
  - Priority-Based Strategy
- 💰 Real slab-based bill calculation
- 📊 Compare savings across strategies
- 🎯 Displays:
  - Units saved
  - New bill amount
  - Money saved
  - Device-wise reduction suggestions

---

## 🧠 Algorithms Used

### 1. Greedy Algorithm
- Prioritizes high energy-consuming appliances
- Quickly reduces units

### 2. Balanced Strategy
- Distributes reduction evenly across all appliances
- More practical for daily usage

### 3. Priority-Based Approach
- Combines efficiency and fairness
- Focuses on impactful devices within limits
